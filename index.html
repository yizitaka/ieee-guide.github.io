Access Guide

Step 1: Connect to Campus Networks

Connect to SEU-WLAN or other campus networks to access IEEE Xplore resources.

Step 2: Visit IEEE Xplore

Go to the official IEEE Xplore website: https://ieeexplore.ieee.org

Off-Campus Access

IEEE Xplore supports both on-campus and off-campus access. For off-campus use, you have two options:

Method 1: VPN Access

  1. Visit the VPN link of your institution
  2. Install Atrust if you haven't installed it
  3. Click "Library Portal"
  4. Click "IEEE"

Method 2: Institutional Sign In

  1. Visit the official IEEE Xplore website
  2. Click "Institutional Sign In"
  3. Click "Access Through institution" and search for "Southeast University"

Search Methods

IEEE Xplore offers multiple ways to search for academic resources. Choose the method that best fits your needs.

Basic Search

Enter keywords in the search box at the top of the page to search. Click the search icon to execute your query.

Fig. 3.1 - Basic Search Interface
Fig. 3.2 - Search Results

Filtering Results

Use the checkboxes at the top of the page to filter results by source type.

Fig. 3.3 - Source Filtering

The available source types include conferences, journals, Early Access Articles, magazines, and books. The number after each option indicates the number of search results.

Use the options on the left side of the page to filter and sort the results by various criteria.

Fig. 3.4 - Filtering Options

You can filter by year, author, affiliation, publication title, publisher, supplemental items, conference location, and publication topics.

Advanced Search

Fig. 3.5 - Advanced Search Interface

Click "Advanced Search" below the single search box, enter keywords, phrases, author names, or query terms in the text boxes, select search fields and operators, and click "Search". The IEEE Xplore platform will retrieve search results.

Command Search

Fig. 3.6 - Command Search Interface

Click "Command Search" to the right of Advanced Search. Enter the search term or method in the text box or select the data fields and operators above the text box to enter the search content. Click "SEARCH" and the IEEE Xplore platform will retrieve the search results.

Citation Search

Fig. 3.7 - Citation Search Interface

Enter keywords, publication details, or author information to perform a citation search.

Author Search

Fig. 3.8 - Author Search Interface

Select "Author Search" from the dropdown next to the Global Search box. Enter the author's first and last name into the fields provided. Click "SEARCH". The author's name will appear as a filter tag at the top under "Active filters". You can click the 'x' icon next to it to remove this filter.

Notice: If multiple authors share the same name, the author's name will appear only once in the cluster analysis section. The number of publications provided in parentheses represents the total count of publications from all authors with that name.

Additional Features

Article Details and Tools

Click the red icon to download the article for more detailed information.

  1. Click "Cite This" to copy or download citations in various formats.
  2. Click the author's name on their details page to explore their other works.
  3. The tags let you quickly browse related document parts.
  4. View similar recommended documents in "More Like This" section.

Download PDFs

Batch download PDF full-text of articles, up to 10 articles at a time with a maximum of 500MB per download.

Export Results

Export CSV format search results, citation information, or manage references in Collaborate platform.

Set Search Alerts

Save search queries and receive notifications for new content.

Search History

Access recent search history, run, modify and combine previous searches.

Practical Advice

Why can't I find results?

Enter technical terms instead of colloquial expressions! Use precise terminology relevant to your field of study.

Why can't I download PDFs?

You should select an institution to access the PDF version. Make sure you're properly authenticated through Southeast University.

Pro Tip: Use Boolean operators (AND, OR, NOT) in your searches to refine results and find more relevant documents quickly.

Search Tips

  • Use quotation marks for exact phrase matching: "machine learning"
  • Use wildcards for partial matching: comput* will find computer, computing, computation
  • Use field-specific searches to narrow results: "Abstract":neural AND "Publication Title":transactions
  • Take advantage of the type-ahead feature that suggests relevant keywords and phrases
